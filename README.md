# Jogos Zerados

Aplicação web simples para registrar e listar seus jogos zerados no navegador.

## Como usar

1. Abra o arquivo `index.html` no navegador.
2. Preencha o formulário com o título, plataforma e outras informações.
3. Clique em "Salvar jogo".
4. Os jogos são armazenados no `localStorage` do navegador e exibidos na lista.

## Recursos

- Adicionar título, plataforma, ano, nota e observações
- Remover jogos individuais
- Limpar todos os registros
- Persistência local no navegador
